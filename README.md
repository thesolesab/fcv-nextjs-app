# FCV — Приложение для футбольных команд и расписания игр

FCV — это современное веб-приложение на Next.js для создания и управления футбольными командами, расписаниями матчей и командными данными через Telegram.

## Возможности

- Создание и редактирование команд
- Управление составом и участниками команд
- Планирование игр и создание расписания матчей
- Просмотр расписания и статуса предстоящих игр
- Интеграция с Telegram для уведомлений и управления командами

## Цель проекта

Проект помогает организовать локальные футбольные турниры, корпоративные матчи или дружеские встречи. Он делает процесс управления командами и расписанием прозрачным и удобным.

## Запуск приложения

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере для просмотра приложения.

## Структура

- `src/app/` — страницы приложения
- `src/components/` — UI-компоненты
- `src/lib/` — вспомогательные модули и API
- `src/services/` — бизнес-логика и работа с данными
- `src/hooks/` — пользовательские React-хуки
- `src/types/` — типы TypeScript

## Технологии

- Next.js
- TypeScript
- Prisma
- Telegram API

## Дальнейшее развитие

Этот проект легко расширять: можно добавить таблицу турниров, систему рейтинга, автоматические уведомления, поддержку нескольких лиг и мобильное приложение.
